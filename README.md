# 乗り打ち収支（Vercel版）

古川・中嶋・坂本の3人で乗り打ち収支を記録し、全員が同じURLで履歴を共有できるWebアプリです。
データはサーバー（Vercel KV）に保存されるので、リロードしても消えず、3人いつでも見返せます。

---

## デプロイ手順（はじめての人向け・全部ブラウザでできます）

### 1. GitHubにアップロード
1. https://github.com で無料アカウントを作成（持っていれば不要）
2. 「New repository」→ 名前を `noriuchi` などにして作成（Privateで可）
3. このフォルダの中身を全部アップロード（「uploading an existing file」からドラッグ&ドロップでOK）

### 2. Vercelにデプロイ
1. https://vercel.com にGitHubアカウントでログイン（無料）
2. 「Add New」→「Project」→ さっきのGitHubリポジトリを選んで「Import」
3. 設定はそのまま「Deploy」を押す（フレームワークは自動でNext.jsと判定されます）

### 3. データ保存先（KV）をつなぐ ← これが「保存できる」ための肝
1. デプロイ後、Vercelのプロジェクト画面で上部「Storage」タブ
2. 「Create Database」→「KV」（または「Upstash for Redis」）を選択 → 作成
3. そのデータベース画面で「Connect Project」→ このプロジェクトを選んで接続
4. 接続すると環境変数（KV_REST_API_URL など）が自動で入ります
5. 「Deployments」タブ → 最新のものの「…」→「Redeploy」で再デプロイ

これで完成。発行されたURL（`https://noriuchi-xxxx.vercel.app`）を3人のLINEに貼れば、全員が同じ履歴を見て入力できます。

---

## ローカルで試す場合（任意）
```
npm install
npm run dev
```
KV未設定だとデータはサーバー再起動で消えます（動作確認用）。本番はKVを必ず接続してください。

## 機能
- 日付・機種（何を打ったか）・3人それぞれの投資額/出玉収入を入力
- 各人の収支とその日の3人合計を自動計算
- 3人で均等に分けた場合の1人あたり分配額を表示
- 通算収支（3人合計・1人あたり）を自動集計
- 過去の履歴をすべて一覧表示・編集・削除可能
- 背景はアップロードされた画像
